# Blog App (React + JSON Server)

A simple blog application with full CRUD functionality (Create, Read, Update, Delete) using React and json-server as a mock backend.

---

## Features

- Read blog list
- Create blogs
- Update blogs (by id)
- Delete blogs (by id)
- Simple authentication
- Mock API using json-server

---

## 📦 Tech Stack

- React (Vite - command: npm create vite@latest)
- JS
- json-server backend (command to save json-server as dev dependency (not included in production): npm install -D json-server)
